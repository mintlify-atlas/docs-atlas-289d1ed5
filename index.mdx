# Welcome to MarkItDown API

MarkItDown API is a powerful FastAPI-based document conversion service that seamlessly transforms documents between formats. Convert PDFs, Word documents, Excel spreadsheets, PowerPoint presentations, and images to clean Markdown, or convert Markdown back to professional DOCX, PPTX, and HTML documents.

## How It Works

MarkItDown API leverages two powerful open-source libraries:

- **MarkItDown**: Microsoft's document-to-Markdown converter with support for 20+ file formats
- **Pandoc**: Universal document converter for generating DOCX, PPTX, and HTML from Markdown

The service provides a clean REST API interface with file upload endpoints, configurable CORS, file size limits, and optional OpenAI integration for enhanced image processing.
