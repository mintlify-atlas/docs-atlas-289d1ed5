## What is MarkItDown API?

MarkItDown API is a powerful REST API that provides bidirectional document conversion:

- **To Markdown**: Convert various document formats (PDF, DOCX, PPTX, Excel, images, and more) into clean, structured Markdown
- **From Markdown**: Transform Markdown files into professional DOCX, PPTX, or HTML documents with custom templates

Built on top of [MarkItDown](https://github.com/microsoft/markitdown) and [Pandoc](https://pandoc.org/), this API makes document conversion as simple as a single HTTP request.

## Use Cases

### Document Processing Pipelines
Extract structured content from PDFs, Word documents, or presentations to process, analyze, or index the text.

### Content Migration
Convert legacy documents to Markdown for modern documentation systems, static site generators, or content management platforms.

### Report Generation
Transform Markdown content (from templates, databases, or user input) into professional DOCX or PPTX reports with custom branding.

### Data Extraction
Convert Excel spreadsheets to Markdown tables for data analysis, transformation, or integration with other systems.

### Accessibility & Archival
Convert various document formats to clean, accessible Markdown or HTML for long-term storage and universal access.

## Architecture

MarkItDown API is built with:

- **FastAPI**: High-performance Python web framework
- **MarkItDown**: Microsoft's document-to-Markdown converter supporting 20+ file formats
- **Pandoc**: Universal document converter (via pypandoc_binary)
- **OpenPyXL**: Excel file processing with multi-sheet support
- **OpenAI** (optional): AI-enhanced image OCR with GPT-4o-mini

<Note>
  The API uses `pypandoc_binary` which bundles Pandoc, so no external dependencies are required.
</Note>

## API Overview

The API exposes four main endpoints:

| Endpoint | Method | Purpose |
|----------|--------|--------|
| `/upload` | POST | Convert files to Markdown |
| `/convert` | POST | Convert Markdown to DOCX/PPTX/HTML |
| `/convert/templates` | GET | Check available default templates |
| `/health` | GET | Health check endpoint |

<Info>
  All file uploads are limited to 50 MB by default (configurable via `MAX_UPLOAD_MB` environment variable).
</Info>

## Response Formats

### Single File Conversion
When converting a single file (or a single-sheet Excel file) to Markdown, the API returns the `.md` file directly:

```
Content-Type: text/markdown; charset=utf-8
Content-Disposition: attachment; filename="document.md"
```

### Multi-File or Multi-Sheet Conversion
When converting multiple files or a multi-sheet Excel file, the API returns a ZIP archive:

```
Content-Type: application/zip
Content-Disposition: attachment; filename="converted_files.zip"
```

For Excel files with multiple sheets, files are named: `{original_name}_{sheet_index}_{total_sheets}.md`

Example: `report_1_3.md`, `report_2_3.md`, `report_3_3.md`
